# Cursor AI Setup Portfolio

A public record of setting up Cursor IDE with Claude Code and Codex extensions.

## Tools Installed

- **Cursor IDE**
- **Claude Code extension** (by Anthropic)
- **Codex extension** (by OpenAI)

## Steps Completed

1. Downloaded and installed Cursor IDE
2. Installed the Claude Code extension via the Extensions marketplace and signed in with my Claude account
3. Installed the Codex extension via the Extensions marketplace and signed in with my ChatGPT account
4. Created this public GitHub repository
5. Cloned it into Cursor and documented the process here

## Issues I Ran Into

- **Cursor vs. Claude desktop app** — Initially confused Cursor (the IDE) with the separate Claude desktop app. They have similar branding, so I had to identify the correct app by its distinct UI (project/folder view vs. chat-only view).
- **Extensions panel shortcut** — `Cmd+Shift+X` didn't respond initially, likely because the sidebar was collapsed. Used **View → Extensions** from the menu bar instead, which worked reliably.
- **Codex activation** — Codex required an open project/folder before it would activate, which aligned naturally with the repo-cloning step.
